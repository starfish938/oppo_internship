# 第二周实习周报

## 一、本周工作内容

本周主要完成了 OFDM 通信系统的基础模块实现，并逐步搭建完整的 OFDM 仿真链路。按照实习计划的任务安排，完成了以下几个部分：

1. **OFDM多径信道建模**
   
   学习并实现了多径信道模型，通过卷积方式模拟信号在信道中的传播过程，并在信道中加入 AWGN 噪声。通过设置不同的信道参数，能够模拟 2径或3径多径传播环境。

2. **循环前缀（CP）机制实验**

   实现了带循环前缀和不带循环前缀的 OFDM 系统仿真，并通过 BER 曲线对比两种情况下系统性能的差异。实验结果表明，在多径信道下加入 CP 可以有效减少符号间干扰（ISI）。

3. **5G NR资源网格与导频设计**

   构建了简化版的 5G NR 资源网格模型，包括：

   - FFT点数：64
   - 有效子载波：52
   - OFDM符号数：14
   - CP长度：8

   同时按照固定间隔插入导频，实现了导频位置与数据位置的区分，并完成资源网格的二维可视化。

4. **OFDM系统完整链路实现**

   在前期模块基础上，实现了完整的 OFDM 收发链路，包括：

   - 随机比特生成
   - QPSK调制
   - 导频插入
   - OFDM调制（IFFT + CP）
   - 多径信道传输
   - OFDM解调（去CP + FFT）
   - 信道估计与频域均衡
   - 符号解调与 BER 计算

5. **系统性能仿真**

   通过设置不同 SNR 条件，对 OFDM 系统进行仿真，并绘制 BER vs SNR 曲线，用于评估系统性能。



## 二、仿真结果

本周实现了多种可视化结果，包括：

- 发送端星座图
- 接收端星座图（加噪声后）
- BER vs SNR 曲线
- OFDM 资源网格可视化

实验结果表明：

- 随着 SNR 提高，BER 逐渐降低
- 在 AWGN 信道下，QPSK 的 BER 曲线基本符合理论趋势
- 多径信道会引入失真，但通过 CP 和均衡可以有效恢复信号



## 三、本周收获

通过本周的实践，加深了对 OFDM 系统结构的理解，包括：

- OFDM 调制与解调原理
- 循环前缀在抗多径干扰中的作用
- 资源网格与导频设计
- 信道建模与频域均衡方法

同时提高了使用 Python 进行通信系统仿真的能力，对数字通信系统的整体流程有了更加清晰的认识。
