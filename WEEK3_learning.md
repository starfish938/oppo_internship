# 第三周周报

本周主要完成了基于 OFDM 系统的信道估计与性能分析任务，具体工作如下：

## 一、工作内容

1. **OFDM 系统与 Rayleigh 信道仿真**  
   - 生成 QPSK 调制符号，并通过 OFDM 调制形成发送信号。  
   - 信号经过 Rayleigh 多径信道和 AWGN 噪声。  
   - 对接收信号进行 OFDM 解调，获得子载波接收数据。  

2. **LS 信道估计及均衡**  
   - 提取导频位置的接收信号，使用最小二乘（LS）方法估计导频子载波的信道响应。  
   - 通过线性插值获得完整子载波信道估计。  
   - 对接收数据进行信道均衡，消除信道衰落对符号的影响。  

3. **性能指标计算与分析**  
   - 计算均衡后的误码率（BER），评估信道估计对系统性能的影响。  
   - 使用归一化均方误差（NMSE）评估信道估计精度，并绘制 NMSE 随信噪比变化的曲线。  
   - 绘制 BER 与 SNR 的关系曲线，直观展示信道估计提升系统性能的效果。  

4. **信道可视化工具开发**  
   - 实现了可同时输出 BER 曲线和 NMSE 曲线的 Python 工具。  
   - 通过工具可快速观察不同 SNR 下 OFDM 系统在 LS 信道估计条件下的性能。  

## 二、实验结果

- BER 曲线显示 LS 信道估计可有效降低误码率，信噪比越高，误码率越低。  
- NMSE 曲线表明 LS 信道估计在 SNR 高于 20dB 时，估计误差达到 -30dB 左右，信道估计精度较好。  
- 实验结果与理论预期一致，验证了信道估计和均衡在 OFDM 系统中的有效性。  

## 三、总结

- 本周完成了 OFDM 信道仿真、LS 信道估计、均衡以及性能分析的完整流程。  
- 通过 BER 和 NMSE 可视化，直观展示了信道估计对系统性能的提升效果。  
- 完成了第三周计划任务，为下一步深入研究信道估计算法优化奠定基础。
